# Anubhav Portfolio 🌌

This portfolio started with a simple idea — to build something that felt personal instead of looking like another copy-pasted developer website.

I wanted the overall vibe to feel futuristic, cinematic, space-inspired, and immersive while still keeping the experience clean and smooth.

A lot of this project was built through experimentation, redesigns, broken layouts, random ideas at night, and continuous hit-and-trial over nearly 2 months.

Some sections were rebuilt multiple times until they finally felt right.

## 🚀 Live Website

https://anubhav1450.github.io/anubhav-portfolio/

## ⚡ Tech Stack

- React
- TypeScript
- Tailwind CSS
- Vite
- Framer Motion

## ✨ Features

- Responsive modern UI
- Space-inspired design system
- Smooth animations and transitions
- Interactive project showcase
- Resume integration
- GitHub Pages deployment

## 📦 Running Locally

```bash
pnpm install
pnpm dev
```
